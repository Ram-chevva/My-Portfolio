Projects

Sitting Posture Detection Using FMCW Audio Signals

This project aims to detect a user's sitting posture using sound waves instead of cameras or wearable sensors. By transmitting FMCW (Frequency-Modulated Continuous Wave) audio signals from a laptop’s speaker and capturing the reflections through a microphone, the system analyzes how sound waves interact with the user's body to classify different postures.

How It Works:

  • A laptop sends FMCW signals via the speaker.
  • The microphone records reflections from the user's body.
  • Signal processing techniques remove noise and interference.
  • K-Means clustering classifies sitting postures.

Key Features:

  • Privacy-focused: No cameras or external sensors required.
  • Fully software-based: Uses only a laptop’s built-in speaker & microphone.
  • Real-time analysis: Detects and classifies postures dynamically.
  • Potential for IoT integration: Can be extended for ergonomic monitoring.
